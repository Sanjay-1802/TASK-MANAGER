# TASK-MANAGER

# About the Project
This is a simple Task Manager application developed using HTML, CSS, and JavaScript. The project helps users manage their daily tasks by allowing them to add, edit, complete, and delete tasks.
I created this project to improve my understanding of JavaScript, DOM manipulation, event handling, and Local Storage. It is a simple and practical application that can be used to organize daily work.

# Live Website
https://sanjay-1802.github.io/TASK-MANAGER/

# Features
* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Store tasks using Local Storage
* Simple and user-friendly interface

# Technologies Used
* HTML
* CSS
* JavaScript

# Project Structure
* index.html – Structure of the application
* style.css – Styling and design
* script.js – Functionality and task management logic

# How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Run the index.html file in any web browser.
